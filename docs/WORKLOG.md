## 2026-06-12 朝 — 新作23番「夜のすきま」

- **体制**: /new-demo の正規手順でオーケストレータが機能ビルダー役を直接実施(検証スクリプト新設・調律3回)→記録係。

  - **構造は削り絵**: 光の層(オフスクリーン: ほの明るい群青の地+PALETTE各色×2の大きなオーロラ塊が漂い呼吸+星90)の上に夜の層(opaque ink)を重ね、指の destination-out ブラシで「すきま」を開ける。はやく削るほどブラシがひろく(15〜64px)。削り口から金の「削りかす」の光が舞う。すきまは自動で癒えて閉じる。起動0.8秒後に小さなお手本のすきまがひらく。
- **調律3回(検証データが導いた)**:
  1. 初版は裏側の光が弱く「くすんだ灰青」→ 塊を大きく・α0.5→0.85、ブラシ開きを0.55→0.85へ。

  3. **決定打: 癒えが速すぎるバグ発見** — 毎フレーム α0.016 の重ね塗りは1秒で約6割閉じる計算で、撮影時には常に半分閉じていた(淡さの真因)。8bit量子化(α<1/255 が捨てられる)を避けつつ遅くするため「**8フレームに1回だけ α0.012 を塗る**」方式に変更 → 20〜30秒かけて閉じる設計どおりの呼吸になり、削った直後はオーロラがあざやかにこぼれるように。
- **検証結果**(新規 `_check/sukima-test.mjs` 5項目 **ALL PASS**・console/pageerror 0): 削り跡の輝度 基準0→**270**(モバイル483)/ 25秒後の癒え 270→156 / HUD / モバイル縦390×844 指削り・はみ出しなし。スクショ `_check/sukima.png`(菫と薄紫と温白のオーロラがこぼれる波形のすきま)/ `sukima-mob.png` をオーケストレータ目視 — 「暗闇に発光」成立。
