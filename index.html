INTERACTIVE STUDY — 10 PROTOTYPES IN A DAY

暗闇に光を灯す、25のインタラクティブ試作。
すべてブラウザで動く叩き台 ＋ 本実装の想定技術つき。

壱 — 3D・AR空間

試作: Three.js ／ 本実装: Unity・Swift
01惑星のすみか
iPadでお絵描きした生き物が、惑星の住人になって歩き出す。描くほどに惑星が賑わうARゲームの原型。
試作: Three.js + Canvas本実装: Unity + AR Foundation (C#)WebSocket同期
02宇宙図書館
幻想的な宇宙の図書館を歩く。透明の階段、浮遊する本、白い妖精の遊び場。部屋を暗くして遊ぶ3D空間。
試作: Three.js 一人称視点本実装: Unity URP / VR (OpenXR)
11光の回廊
闇に灯る無限のアーチをくぐり、光の尾を引いて進む。長押しで加速、視線は指に揺れる。
試作: Three.js本実装: Unity URP / VR (OpenXR)
16クラゲの天蓋
暗い海の底、頭上に漂う発光の群れ。触れれば光が連鎖する。ドラッグで深海を見回す。
試作: Three.js本実装: Unity URP / VR (OpenXR)
19光の庭
夜の丘に種を描いて植える、育てて残せる庭。蛍が集まり、音が生まれ、保存した庭はまた咲く。10人がかりの旗艦作。
試作: Three.js modules本実装: Unity URP (C#)Web AudiolocalStorage
22霧の灯台
ドラッグで灯台の光をまわすと、霧の沖の小さな舟が気づいて、港へ帰ってくる。あなたは光の番人。
試作: Three.js本実装: Unity URP / VR (OpenXR)

弐 — 身体・音センシング

試作: Canvas + Webカメラ・マイク ／ 本実装: Kinect・TouchDesigner
03息で咲く光の花野
マイクに息を吹きかけると花畑がざわめき、花びらが光の粒になって舞う。静けさの中で再び芽吹く。
試作: Canvas + Web Audio本実装: TouchDesigner + マイク
05群れと踊る鏡
カメラがあなたの動きを捉え、光の蝶の群れが集まり、驚いて散る。体ぜんぶがコントローラー。
試作: Canvas + Webカメラ差分本実装: Unity + Azure Kinect / MediaPipe
07音紋の池
手を叩くと暗い水面に波紋がひろがり、光の魚が生まれる。音を重ねるほど池が満ちていく。
試作: Canvas + Web Audio本実装: Max/MSP + TouchDesigner
12声の星雲
声の大きさと高さで膨らみ色づく星雲。黙れば静かに眠る。マイクがなくても指の動きが声になる。
試作: Canvas + Web Audio本実装: Max/MSP + TouchDesigner
20こだまの洞窟
闇の奥へ呼びかけると、こだまが光の波紋になって遅れて帰ってくる。返るたびに淡く、深く。
試作: Canvas 2D + Web Audio本実装: Max・MSP + 投影
24夜のしゃぼん
息を吹きこむと、光のしゃぼん玉がふくらんで夜へ漂う。ふれると、ほどけて星屑になる。
試作: Canvas + Web Audio本実装: TouchDesigner + マイク
26しずかの森
指をそっと置いて、動かないでいる。しずけさに、臆病な光たちがなついて寄ってくる。うごくと、にげてしまう。
試作: Canvas 2D（静止検出）本実装: Azure Kinect（静止検出）+ 投影

参 — タッチ・ドローイング

試作: Canvas 2D ／ 本実装: openFrameworks・Unity + タッチウォール
04一筆書きの精霊
指で描いた一筆書きの線が、そのまま生き物になって泳ぎ出す。線の速さと曲がりで性格が変わる。
試作: Canvas 2D本実装: openFrameworks (C++)
06言葉の雨
文字が雨のように降る。触れた言葉は弾けて、その意味する風景——雪、花、火——に変わる。
試作: Canvas 2D本実装: Unity + タッチウォール (C#)
08光の砂庭
発光する砂の禅庭。なぞると砂紋が描かれ、紋様の交差から光の虫が生まれる。やがて風が紋を消す。
試作: Canvas 2D本実装: TouchDesigner + 赤外タッチ
13光の糸琴
張った光の糸をなぞれば、弦が震えて粒をこぼし小さく鳴る。糸はやがて闇に溶ける。
試作: Canvas 2D + Web Audio本実装: openFrameworks (C++)
14ふれると灯る街
ふれた窓から灯がともり、人影と星が増える夜の街。放っておくと灯りはひとつずつ眠る。
試作: Canvas 2D本実装: Unity + タッチウォール (C#)
17水鏡の文字
夜に描いた光の文字が水面に揺らぎ、粒となって波紋に還る。波が収まれば水面はまた鏡になる。
試作: Canvas 2D本実装: openFrameworks (C++)
23夜のすきま
指で夜を削ると、むこうがわの光がこぼれる。すきまは、ゆっくりと閉じていく。
試作: Canvas 2D本実装: openFrameworks (C++) + タッチウォール

肆 — マルチデバイス連携

試作: Canvas（1画面で擬似再現）／ 本実装: Swift + WebSocket + プロジェクター
09星座を紡ぐ夜
星と星をなぞって繋ぐと、その形から星座の獣が生まれて夜空を駆ける。みんなの獣が同じ空に住む。
試作: Canvas 2D本実装: Unity + スマホ連携 (WebSocket)
10影に咲く花火
スワイプで光の種を打ち上げると花火が咲き、残光が地に落ちて花畑として積もっていく。
試作: Canvas + スワイプ本実装: Swift (加速度) + WebSocket + 投影
15渡り鳥の手紙
スワイプから生まれた光の鳥が境界を越え、とおくの空で星の手紙になる。ふたつの端末をひとつの夜が繋ぐ。
試作: Canvas 2D（擬似2画面）本実装: Swift + WebSocket + 投影
18灯籠流しの川
触れて灯した灯籠を川へ送ると、とおくの岸辺に金色が並ぶ。流すほど川がきらめく。
試作: Canvas 2D（擬似2画面）本実装: Swift + WebSocket + 投影
21振り子の夜
三台の端末に吊られた光の振り子。ばらばらに乱しても、見ているうちにひとつの呼吸へそろってゆく。
試作: Canvas 2D（1画面で擬似再現）本実装: Swift (加速度) + WebSocket + 投影
25月のみちしお
ふたつの端末は、それぞれ小さな海。月を渡すと、月の下の海が満ちて、とおい海は引いていく。
試作: Canvas 2D（1画面で擬似再現）本実装: Swift (加速度) + WebSocket + 投影
